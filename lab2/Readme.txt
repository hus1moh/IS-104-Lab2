Det er 3 filer i lab 2, html, css og javacript!
Kontrollspørsmål:
Her er det de tre filer, CSS, JS og HTML som er inplementert i min appliksjon. 
Model: Html filen, siden det er denne filen som er modellen og holder på dataen vi har.

View: CSS Filen er denne som bestemmer hvordan vår data skal vises og hvor de skal plasseres. 
Controller: Javascript Filen er den som kan kontrollere hva som skal vises av informasjonen i HTML filen. 
Poenget er å redusere gjentagelse av data i tilleg til at det gjør mer oversiktig
